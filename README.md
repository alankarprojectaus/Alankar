This repository presents a final year project focused on Multilabel Alankar Classification in Hindi Poetry using Deep Learning. The project aims to automatically identify and classify multiple Alankars (poetic devices) present within Hindi poems.
To achieve this, two deep learning models have been developed:
IndicFusion – a hybrid model designed to capture rich contextual and linguistic features from Hindi text
TF-POS Net – a model leveraging TF-IDF and Part-of-Speech (POS) features for effective multilabel classification
Both models are designed to detect multiple Alankars simultaneously from a single poem, enabling a more comprehensive analysis of poetic structures in Hindi literature.
